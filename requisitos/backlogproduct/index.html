    Histórias de Usuários

    E01 Gerenciamento de Usuários

    E01FE01 Manter Usuários

    ID História de Usuário Pontos
    US01 Eu, como usuário, desejo realizar login no sistema para ter acesso às minhas funcionalidades no sistema.
    US02 Eu, como usuário, desejo alterar meus dados para atualizá-los e corrigí-los.
    US03 Eu, como usuário, desejo alterar a minha senha para manter a segurança dos meus dados.
    US04 Eu, como administrador, desejo cadastrar novos usuários para que os membros e alunos do projeto tenham acesso ao sistema.
    US05 Eu, como administrador, desejo alterar as permissões de um usuário para mantê-las alinhadas com a sua função no sistema.
    US06 Eu, como administrador, desejo remover um usuário do sistema para que um membro desligado do projeto não tenha mais acesso ao sistema.
    US07 Eu, como membro do projeto, desejo ver o perfil de outros usuários do sistema para que eu possa ter acesso aos seus dados.
    US08 Eu, como usuário, desejo ver o meu perfil para que eu possa verificar os meus dados.
    US09 Eu, como membro do projeto, desejo listar os membros para ter acesso a todos os membros cadastrados no sistema.
    US10 Eu, como membro do projeto, desejo listar os alunos para ter acesso a todos os alunos cadastrados no sistema.

    E01FE02 Controlar Frequência

    ID História de Usuário Pontos
    US11 Eu, como administrador, desejo registrar informações de frequência dos usuários para que haja controle das faltas dos membros e alunos do projeto.
    US12 Eu, como usuário, desejo ter acesso às minhas informações de frequência para que eu possa ter controle das minhas faltas.
    US13 Eu, como membro do projeto, desejo ter acesso às informações de frequência dos alunos para que haja monitoramento das faltas dos alunos.
    US14 Eu, como membro do RH, desejo ter acesso às informações de frequência de todos os membros para que haja monitoramento das faltas dos membros.
    US15 Eu, como usuário, desejo justificar falta para informar o motivo da minha ausência em uma atividade do projeto.
    US16 Eu, como membro do RH, desejo ter acesso às justificativas de falta dos membros para que eu saiba o motivo da ausência dos membros em suas atividades do projeto.
    US17 Eu, como diretor do projeto, desejo ter acesso às justificativas de falta dos membros para que eu saiba o motivo da ausência dos membros em suas atividades do projeto.

    E02 Gerenciamento de Notícias

    E02FE01 Manter Notícias

    ID História de Usuário Pontos
    US18 Eu, como usuário, desejo ver uma lista de notícias que corresponde a minha função no sistema para que eu esteja informado sobre as notícias que cabem a mim.
    US19 Eu, como usuário, desejo ter uma visão detalhada de uma notícia escolhida na lista de notícias para que eu possa ler a notícia por completo.
    US20 Eu, como diretor do projeto, desejo publicar notícias para que os outros usuários possam ver os informes que lhes competem.
    US21 Eu, como membro do Marketing, desejo publicar notícias para que os outros usuários possam ver os informes que lhes competem.
    US22 Eu, como diretor do projeto, desejo editar uma notícia de minha autoria para mantê-la atualizada e corrigida.
    US23 Eu, como membro do Marketing, desejo editar uma notícia de minha autoria para mantê-la atualizada e corrigida.
    US24 Eu, como administrador, desejo excluir uma notícia para dispensar notícias que foram postadas equivocadamente.

    E02FE02 Alertar Notícias

    ID História de Usuário Pontos
    US25 Eu, como usuário, desejo ser alertado por e-mail sobre uma nova notícia que me compete para saber que uma nova notícia foi publicada instantaneamente.
    US26 Eu, como autor de notícia, desejo que um alerta da minha notícia publicada seja enviada por e-mail aos interessados para que eles saibam que há uma nova notícia instantaneamente.

    E03 Gerenciamento de Disciplinas

    E03FE01 Manter Disciplinas

    ID História de Usuário Pontos
    US27 Eu, como administrador, desejo criar uma disciplina para que os materiais desta disciplina estejam contidos dentro dela.
    US28 Eu, como administrador, desejo criar uma frente para organizar a disciplina de acordo com as suas divisões.
    US29 Eu, como administrador, desejo editar uma disciplina para que ela esteja atualizada e corrigida.
    US30 Eu, como administrador, desejo editar uma frente para que ela esteja atualizada e corrigida.
    US31 Eu, como administrador, desejo remover uma disciplina para que uma disciplina que esteja fora do contexto do projeto não esteja mais cadastrada no sistema.
    US32 Eu, como administrador, desejo remover uma frente para que uma frente que esteja fora do contexto do projeto não esteja mais cadastrada no sistema.
    US33 Eu, como usuário, desejo ver uma tela com as disciplinas ministradas para ter acesso aos seus materiais.

    E03FE02 Manter Materiais

    ID História de Usuário Pontos
    US34 Eu, como usuário, desejo ver uma tela com os materiais das frentes para baixar os materiais.
    US35 Eu, como professor, desejo publicar um material da minha disciplina para que os alunos possam baixá-lo.
    US36 Eu, como membro do Conteúdo, desejo publicar um material em uma disciplina para que os alunos possam baixá-lo.
    US37 Eu, como autor de material, desejo apagar um material para dispensar um material publicado equivocadamente.
